## 🎯 O que você vai ter ao final

✅ **Sistema funcional de gestão financeira**

```
Você envia no Telegram:
"-lanche 25"
        ↓
Bot confirma e registra
        ↓
Aparece no dashboard em tempo real
        ↓
Você pode filtrar por categoria,
ver saldo, despesas, receitas
```

✅ **Hospedado no Vercel (gratuito)**
- URL pública: https://seu-app.vercel.app
- Auto-deploy ao fazer push no GitHub
- Logs e monitoramento

✅ **Banco de dados PostgreSQL (Supabase)**
- Dados seguros
- Histórico completo
- Consultas rápidas

✅ **Bot Telegram 24/7**
- Acesse de qualquer lugar
- Registre gastos na hora
- Respostas instantâneas

